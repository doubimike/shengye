盛业资本荣获“中国商业保理行业创新奖“

3月28-31日，由中国服务贸易协会商业保理专业委员会主办的第五届(2017)中国商业保理行业峰会在天津滨海新区隆重举行。本届峰会是中国商业保理行业峰会作为我国应收账款融资领域规模最大的年度盛会，盛业资本旗下的盛业商业保理有限公司（以下简称盛业保理）受邀出席，并荣获“第二届(2017)中国商业保理行业创新奖“。

国际保理商联合会前秘书长Jeroen Kohnstamm为获奖单位颁发“中国商业保理行业创新奖“

中国商业保理行业峰会作为我国应收账款融资领域规模最大的年度盛会，围绕“商业保理可持续发展之道”的主题，深入探讨了如何在新形势下充分发挥商业保理这一被国际经验证明具有逆经济周期特点、最适合成长型中小企业贸易融资工具的独特作用，通过盘活企业存量应收账款、加速账款回收、破解中小企业融资难融资贵和降低大企业杠杆率的有效途径。

两会期间，李克强总理主持召开国务院常务会议，确立金融支持工业增效升级的措施，首次提出要大力发展应收账款融资。作为中国商业保理行业年度盛会，“中国商业保理行业峰会”连续五届的成功举办不仅促进了商业保理行业内的沟通与交流，更是对整个行业的规范、创新与发展起到了推动作用。

峰会现场

据悉，中国商业保理行业峰会是我国商业保理行业规模最大的年度盛会，已成功举办五届。在本届峰会上，“第二届(2017)中国商业保理行业创新奖”名单同步揭晓，国内领先的商业保理服务商“盛业保理” 凭借“产融结合”的创新金融模式成功获得本次奖项。

在本届峰会论坛上，盛业资本副总经理陈仁泽与参会嘉宾分享了盛业保理服务中小企业的创新理念与实践成果。

对此，副总经理陈仁泽表示，盛业保理金融方面的服务主要专注于创新类金融，金融资本创新与价值的结合。盛业保理将在保理方面，争取为企业打造一个提升融资的渠道。

盛业资本副总经理陈仁泽分享创新案例

盛业资本以深圳为运营总部，全面布局华北、华东、华南并建立沿海服务网络，是国内少数的独立外资金融投资企业。盛业资本逐渐成长，形成了立足产业金融，业务主要涉及能源、建筑及医疗三个行业，服务客户主要以大型国有企业、上市公司为核心厂商的供应商或经销商的商业拓展模式。盛业资本旗下的盛业商业保理有限公司成立于2013年底，是国内首批商业保理公司之一。截至目前，盛业保理已经和国信证券、中证信用、招商银行、浦发银行、包商银行等多家金融机构合作，共同构建以合作共赢为核心的新型金融合作关系。

业内分析人士认为，在目前中央大力发展应收账款融资，破解中小企业融资难的政策导引下，商业保理将成为全球金融服务方式变革的主要推动者，推进保理业务将会带动整个金融、支付、信用的大力发展。未来如盛业保理这样的企业，有望借势政策红利，充分发挥金融机构、第三方物流及供应链上下游企业的优势，从产业链角度解决中小企业融资难问题，促进中小金融机构和企业的发展，提高产业链竞争力，实现多方共赢。全力挖掘商业保理的价值蓝海，引领商业保理行业平稳、持续的发展。
