大事记

2014年 开展业务
  • 成为中国服务贸易协会商业保理专业委员会成员(该委员会是获国务院批准成立的非牟利国家社会机构，由商务部规管及向民政部登记)
  • 成为天津第一家成功引进外债的保理公司
  • 通过招商银行成功发行保理行业第一笔“专项资产管理计划”4.5亿人民币
2015年
  • 天津市商业保理协会会员单位
  • 成为首家与阿里巴巴平台招财宝合作的保理公司
2016年
  • 通过包商银行成功发行委贷产品“保理信托计划”1亿人民币
  • 公司累计转让金额总量达110亿人民币
  • 成为天津自贸区东疆第一家引进最大一笔离岸人民币外债的保理公司
  • 受邀成为深圳市商业保理协会副会长单位
  • 通过中国供销集团旗下普惠金融平台进行应收账款卖断，共卖断AR金额总计4.6亿人民币
  • 受邀参加在南非举办的国际保理商联合会（FCI）
  • 通过国信证券成功发行“专项资产管理计划”4000万人民币
  • 受邀成为中国服务贸易协会商业保理专业委员会副主任单位
  • 为客户量身打造的首家使用电子签章的线上商业保理融资系统“盛易通”正式上线
